アプリについて

Offline Japan Mapは、国土地理院のベクトルタイル提供実験を利用し、日本の地図を事前に保存して確認できるiOSアプリです。保存地図、登録地点、GPXファイル、バックアップを使い、通信が不安定な場所での地図確認を補助します。国土地理院の公式アプリではありません。

問い合わせ

メール: tenkarars@gmail.com

地図が表示されない

オンライン地図表示には通信が必要です。通信なしで使う場合は、事前に保存地図を作成してください。

現在地が表示されない

iOSの設定で位置情報サービスと、このアプリの位置情報許可を確認してください。Simulatorでは位置情報の設定により現在地が表示されない場合があります。

保存地図の外になった

保存地図の枠を確認し、必要に応じて範囲を広げて再保存してください。通信なし運用では保存地図内の地図だけを利用できます。

バックアップで保存されるもの

登録地点と保存地図の情報をあとで戻せるようにします。

GPXファイルを読み込む

GPXファイルを読み込むと、歩いた記録や歩く計画を地図上に表示できます。登録地点が含まれるGPXファイルでは、登録地点として追加できる場合があります。この機能では、記録、自動ナビ、ルート外れ警告は行いません。
