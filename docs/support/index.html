アプリについて

Field Vector Mapは、国土地理院のベクトルタイル提供実験を利用したフィールド向け地図アプリです。保存済み範囲、POI、JSONバックアップを使い、通信が不安定な場所での地図確認を補助します。国土地理院の公式アプリではありません。

問い合わせ

メール: tenkarars@gmail.com

地図が表示されない

オンライン地図表示には通信が必要です。通信なしで使う場合は、事前に保存済み範囲を作成してください。

現在地が表示されない

iOSの設定で位置情報サービスと、このアプリの位置情報許可を確認してください。Simulatorでは位置情報の設定により現在地が表示されない場合があります。

保存済み範囲外になった

保存済み範囲の輪郭を確認し、必要に応じて範囲を広げて再保存してください。通信なし運用では保存済み範囲内の地図だけを利用できます。

JSONバックアップで保存されるもの

POIと保存地図メタデータのみ保存します。地図タイル本体はJSONバックアップには含まれません。
